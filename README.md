# Ship Shit Show — Opus 4.8

A collection of demos built live on stream with Claude Code (Opus 4.8).

📺 **Stream:** https://www.youtube.com/watch?v=p-WXHu2gU2s

## Demos

| Directory | Project | What it is |
|-----------|---------|------------|
| [`agentsdemo`](./agentsdemo) | **AgentFlow Debugger** | A React Flow–based debugger for visualizing agent execution flows. |
| [`cademo`](./cademo) | **PocketCAD** | A browser CAD tool for laser-cut / flat parts with live 2D + 3D preview and SVG/DXF export. |
| [`fpsdemo`](./fpsdemo) | **FPS Arena** | A Three.js first-person shooter demo. |

Each demo is a standalone React + TypeScript + Vite app.

## Running a demo

```bash
cd <demo-directory>
npm install
npm run dev
```
